RCS Examples for Intel Xeon Phi Knights Landing (KNL)

Description of files

  • pi_openmp_knl.c - An OpenMP code to compute the value of pi.
  • script.openmp - A sample job script to run OpenMP programs.
  • pi_mpi_knl.c - An MPI code to compute the value of pi.
  • script.mpi - A sample job script to run MPI programs.
